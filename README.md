# Carbon Emissions Dashboard

A visual analytics dashboard for exploring trip data and carbon emissions.

![Dashboard Demo](dashboard-demo.gif)

## Overview

This project visualizes transportation data to help understand carbon emission patterns using interactive maps and data tables.

## Features

- **Interactive Map**: Groups trip data by purpose and mode of transportation.
- **Data Visualization**: Displays carbon emissions and trip details.
- **Granular Views**: Zoom in to see individual trip segments and specific transportation modes.

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Data**: CSV processing (Emissions & Trip Data)

## Getting Started

1. Clone the repository.
2. Open `index.html` in your web browser.
